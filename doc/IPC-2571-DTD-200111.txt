IPC 2571 Document Type Definition (DTD)
Generic Requirements for Supply Chain Communication of Product Data Exchange (PDX)

From: http://webstds.ipc.org/2571pub.pdf
See: http://xml.coverpages.org/pdx.html

"It is expected that a  future version of Product Data
eXchange XML definitions will be written in XML Schema.
The following is a master DTD that includes all elements from the 
IPC-2571, IPC-2576 and IPC-2578."

--------------------------------------------------------------
